ClipForge

Build zero-cost short-form clips from long videos with local tools. Use this page to generate the exact CLI command, keep your setup checklist close, and move faster when you're batch-processing podcasts, interviews, or educational videos.

Command Builder

python pipeline.py "YOUR_SOURCE"

Mode: Ollama local inference

Quick Run

1. Start Ollama
ollama serve
2. Verify Setup
python setup_check.py
3. Run Pipeline
python pipeline.py "URL_OR_PATH"
4. Open Output
outputs/clips/

Checklist

  • Install Python 3.10+ and Git.
  • Install FFmpeg and confirm it is on PATH.
  • Install Ollama and pull at least one local model.
  • Run pip install -r requirements.txt.
  • Optional: install WhisperX for better timestamps.
  • Optional: add a Gemini API key near the top of pipeline.py.

Troubleshooting

  • ffmpeg not found: reinstall it and restart the terminal.
  • Ollama unreachable: make sure ollama serve is still running.
  • No module named whisper: rerun pip install -r requirements.txt.
  • Bad YouTube downloads: update yt-dlp with pip install -U yt-dlp.
  • No clips returned: widen your min and max clip duration.
